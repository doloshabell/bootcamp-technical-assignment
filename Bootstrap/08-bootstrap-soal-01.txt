1. Apa itu CSS Framework?
2. Apakah bootstrap satu-satunya CSS Framework yang dapat kita gunakan?
3. Sebutkan beberapa CSS Framework selain Bootstrap
4. Bagaimana tata cara penggunaan Bootstrap?

Answer

1. CSS Framework adalah hasil codingan yang menjadi kerangka kerja untuk memudahkan developer untuk mendesain website
2. Bootstrap bukanlah satau-satunya CSS Framework
3. Foundation, Bulma, Semantic UI, Tailwind, Skeleton, PureCSS, Miligram, Materialize CSS, UIKit, Blueprint, Gumby
4. Kita dapat langsung menuju link Bootstrap Documentation -> https://getbootstrap.com/ , lalu kita bisa bebas mencari 
style atau desain untuk kebutuhan website yang akan dibuat.
